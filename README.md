# Pi-HOC: Pairwise 3D Human-Object Contact Estimation

This repository contains the official code implementation for the paper:

> **Pi-HOC: Pairwise 3D Human-Object Contact Estimation**
> Sravan Chittupalli, Ayush Jain, Dong Huang

[**Paper**](https://arxiv.org/abs) | [**Project Page**](https://pi-hoc.github.io/)

---

## Overview

Pi-HOC is a fast, single-pass framework for dense 3D human–object contact prediction that scales efficiently to complex, multi-human scenes. By representing every human-object pair with dedicated tokens and refining them through InteractionFormer, it produces accurate, instance-level contact maps on SMPL meshes. Compared to prior methods, Pi-HOC delivers significantly better localization and accuracy while achieving up to 20× higher inference throughput, making it a practical solution for real-world interaction understanding.
